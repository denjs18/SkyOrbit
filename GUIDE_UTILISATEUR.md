## Introduction

ULM Gestion Club est une plateforme complète de gestion pour les associations d'ULM. Elle vous permet de :

- Gérer les réservations de créneaux de vol
- Suivre les conditions météorologiques en temps réel
- Administrer votre flotte d'ULM
- Gérer vos membres et leurs qualifications
- Organiser et vendre des baptêmes de l'air
- Suivre votre comptabilité
- Communiquer avec vos membres via les actualités

## Connexion

### Première Connexion

1. Accédez à l'URL de votre installation : `https://votre-domaine.fr`
2. Entrez vos identifiants (email ou nom d'utilisateur et mot de passe)
3. Cliquez sur "Décoller !"

### Comptes de Démonstration

Pour tester l'application, utilisez ces comptes :

- **Administrateur** : `admin@ulm-club.fr` / `admin123`
- **Instructeur** : `instructeur@ulm-club.fr` / `inst123`
- **Pilote** : `pilote@ulm-club.fr` / `pilot123`

⚠️ **Changez immédiatement ces mots de passe en production !**

### Mot de Passe Oublié

1. Cliquez sur "Mot de passe oublié ?"
2. Entrez votre email
3. Suivez les instructions reçues par email

## Réservations

### Créer une Réservation

1. **Via le Dashboard** : Cliquez sur "Actions rapides" > "Nouvelle réservation"
2. **Via la page Réservations** : Bouton "Nouvelle réservation"
3. **Via le Calendrier** : Cliquez sur une date

**Remplir le formulaire** :
- Machine (obligatoire)
- Pilote (obligatoire)
- Date et heures (début/fin)
- Type de vol : Vol libre, Formation, Baptême, Maintenance
- Instructeur (si formation)
- Remarques

### Visualiser les Réservations

**3 Vues Disponibles** :

1. **Vue Calendrier**
   - Vue mensuelle/hebdomadaire/journalière
   - Cliquez sur un événement pour voir les détails
   - Code couleur par statut

2. **Vue Liste**
   - Tableau détaillé de toutes les réservations
   - Filtres par machine, statut, pilote
   - Recherche par nom

3. **Vue Planning**
   - Timeline par machine
   - Vision claire des disponibilités
   - Idéal pour éviter les conflits

### Modifier ou Annuler

- Cliquez sur une réservation
- Utilisez les boutons "Modifier" ou "Annuler"
- Confirmation demandée pour annulation

### Filtres

Affinez votre recherche par :
- Machine spécifique
- Statut (confirmée/en attente/annulée)
- Nom du pilote

## Météo

### Conditions Actuelles

**Informations Principales** :
- Température (et ressenti)
- État du ciel
- Conditions de vol globales

**Détails** :
- 💨 Vent : Vitesse, direction, rafales
- 💧 Humidité et point de rosée
- 📊 Pression atmosphérique
- 👁️ Visibilité
- ☁️ Couverture nuageuse et plafond
- ☀️ Indice UV
- 🌅 Heures lever/coucher du soleil

### Évaluation des Conditions de Vol

Le système évalue automatiquement les conditions :

- ✅ **Excellentes** : Tous les paramètres sont optimaux
- 👍 **Bonnes** : Vol possible, quelques réserves
- ⚠️ **Marginales** : Vol déconseillé pour débutants
- ❌ **Mauvaises** : Vol fortement déconseillé

### Prévisions 7 Jours

- Températures min/max
- Conditions météo prévues
- Force et direction du vent
- Planifiez vos vols à l'avance

### METAR et TAF

Pour les pilotes confirmés :
- **METAR** : Observations météo aéronautiques actuelles
- **TAF** : Prévisions aéronautiques
- Décodage en français fourni

### Actualisation

- Actualisation automatique toutes les 5 minutes
- Bouton "Actualiser" pour mise à jour manuelle

## Gestion des Machines

### Vue d'Ensemble

Liste complète de votre flotte avec :
- Immatriculation
- Type d'ULM (pendulaire, multiaxe, autogire, etc.)
- Statut (disponible, en vol, maintenance, indisponible)
- Heures de vol totales
- Prochaine révision

### Ajouter une Machine

1. Cliquez sur "Ajouter une machine"
2. Renseignez :
   - Immatriculation
   - Marque et modèle
   - Type d'ULM
   - Année de fabrication
   - Heures de vol actuelles
   - Intervalles de maintenance

### Suivi de Maintenance

**Fonctionnalités** :
- Carnet de maintenance numérique
- Rappels automatiques
- Historique des interventions
- Documents techniques (manuel, certificat de navigabilité)

### Disponibilité

Marquez une machine comme :
- ✅ Disponible
- 🔧 En maintenance
- ⛔ Indisponible
- ✈️ En vol

Les machines non disponibles n'apparaissent pas dans les réservations.

## Gestion des Membres

### Liste des Membres

Visualisez tous vos membres avec :
- Nom et prénom
- Rôle (Admin, Instructeur, Pilote, Membre)
- Statut (Actif, Inactif, Suspendu)
- Dernière connexion
- État de la cotisation

### Ajouter un Membre

1. Cliquez sur "Ajouter un membre"
2. **Informations personnelles** :
   - Nom, prénom
   - Email, téléphone
   - Adresse
   - Date de naissance

3. **Informations aviation** :
   - Numéro de licence
   - Qualifications (pilote, instructeur, etc.)
   - Date de validité de la licence
   - Visites médicales

4. **Informations association** :
   - Date d'adhésion
   - Montant de la cotisation
   - Statut de paiement

### Profil Membre

Chaque profil contient :
- Photo
- Coordonnées complètes
- Historique des vols
- Statistiques (heures de vol, nombre de vols)
- Documents (licence, certificat médical, assurance)
- Notes et remarques

### Rôles et Permissions

**Administrateur** :
- Accès complet
- Gestion des utilisateurs
- Configuration du système

**Instructeur** :
- Réservations
- Validation des vols
- Suivi des élèves

**Pilote** :
- Réservations personnelles
- Consultation météo
- Consultation du planning

**Membre** :
- Consultation uniquement
- Actualités

## Baptêmes de l'Air

### Configuration des Formules

**3 Formules Pré-configurées** :

1. **Découverte (70€)** - 15 minutes
   - Survol de la région
   - Photo souvenir

2. **Sensation (120€)** - 30 minutes
   - Prise en main
   - Photos + Vidéo

3. **Prestige (200€)** - 60 minutes
   - Pilotage accompagné
   - Pack photos/vidéos complet
   - Certificat dédicacé

Vous pouvez modifier ces formules dans Paramètres.

### Gérer les Réservations de Baptêmes

**Vue d'ensemble** :
- Baptêmes planifiés
- En attente de validation
- Effectués
- Annulés

**Nouvelle Réservation** :

1. **Informations Client** :
   - Nom, prénom
   - Contact (téléphone, email)
   - Nombre de passagers (1-2)
   - Poids total (important !)

2. **Réservation** :
   - Formule choisie
   - Date et heure souhaitées
   - Pilote assigné

3. **Paiement** :
   - Statut (en attente, payé, acompte)
   - Moyen de paiement
   - Référence de transaction

### Actions sur un Baptême

- ✅ Confirmer
- ✈️ Marquer comme effectué
- ❌ Annuler
- 📧 Envoyer confirmation par email
- 🔔 Envoyer un rappel
- 🧾 Générer une facture

### Page Publique de Réservation

Un formulaire public permet au grand public de réserver :

- Accessible depuis : `baptemes-public.html`
- Intégrable sur votre site web via iframe
- Formulaire de contact pour demande de réservation
- Présentation des formules

**Code d'intégration** :
```html
<iframe src="https://votre-site.fr/baptemes-public.html"
        width="100%" height="800"></iframe>
```

### Statistiques

- Nombre de baptêmes par mois/année
- Revenus générés
- Formules les plus populaires
- Taux de satisfaction (si sondage activé)

## Conseils et Bonnes Pratiques

### Sécurité

1. **Changez les mots de passe par défaut**
2. **Utilisez HTTPS** en production
3. **Sauvegardez régulièrement** votre base de données
4. **Limitez les accès administrateur**

### Performance

1. **Archivez les anciennes réservations** (> 1 an)
2. **Optimisez les images** uploadées
3. **Videz le cache** régulièrement

### Organisation

1. **Planifiez les réservations à l'avance**
2. **Vérifiez la météo quotidiennement**
3. **Tenez à jour le carnet de maintenance**
4. **Communiquez via les actualités**

### Support

Pour toute question :
- Consultez la documentation complète
- Contactez le support : support@ulm-gestion-club.fr
- Rejoignez la communauté d'utilisateurs
